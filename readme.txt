SEO Book WordPress NewsPress is a fully customizable blank WordPress theme designed for news, blogs, and article websites. All customizations are made via the default WordPress Customizer.

== Description ==

SEO Book WordPress NewsPress is a lightweight and fully customizable WordPress theme perfect for news, blogs, and article websites. This theme is built with SEO best practices in mind to help your site rank better. It is responsive, fast-loading, and widget-ready, offering a seamless experience on any device. The theme is also compatible with the Gutenberg block editor and supports easy integration with social media platforms.

== Installation ==

1. Download the theme from the releases section.
2. Upload the theme files to the /wp-content/themes/ directory.
3. Activate the theme through the 'Appearance > Themes' menu in WordPress.
4. Customize your theme via the WordPress Customizer.
